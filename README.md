## 项目简介
本项目是一款基于Web的自习室预约管理系统，面向高校学生与备考人群，提供座位查询、在线预约、时段选择、签到签退、违规记录等核心功能。用户可按需预约座位，系统自动管控使用时长与占用状态，避免占座、抢座等问题。

同时配备管理后台，支持座位管理、订单审核、数据统计与用户权限控制。系统旨在规范自习室使用秩序，提升空间利用率，为用户提供便捷、公平、高效的自习环境，实现自习室管理数字化与智能化。
