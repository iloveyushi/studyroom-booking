## 遇到的问题和解决
1. 问题：Git 提交时报错 remote origin already exists
解决：使用 git remote rm origin 清除旧仓库，重新绑定新仓库后推送成功。

2. 问题：本地运行 FastAPI 出现模块找不到 app
解决：使用 python -m app.main 启动，成功解决模块路径问题。

3. 问题：Docker 构建后服务无法启动
解决：修正 Dockerfile 工作目录与启动命令，确保端口映射正确。
