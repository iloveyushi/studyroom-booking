# 架构设计文档

## 一、前端架构说明

本系统前端按照使用角色划分为**用户端**和**管理端**两个子系统。

- **用户端**主要面向普通用户，提供：
  - 自习室查询
  - 座位预约
  - 个人信息管理

- **管理端**主要面向系统管理员，提供：
  - 用户管理
  - 座位管理
  - 预约管理

由于两类用户在功能与权限上存在明显差异，因此在前端架构设计中对其进行模块化划分，并分别设计页面结构，以提高系统整体结构的清晰性与可维护性。
