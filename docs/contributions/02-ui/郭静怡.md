## 遇到的问题和解决
1. 问题：首页统计卡片颜色区分不够明显，容易混淆。
   解决：调整卡片配色方案，采用主色、辅助色、强调色三类色彩进行区分，提升信息可识别性。

2. 问题：表格字段较多，在小屏设备上显示不全。
   解决：设计响应式表格，支持横向滚动，并优化字段排列顺序，优先展示关键信息。
