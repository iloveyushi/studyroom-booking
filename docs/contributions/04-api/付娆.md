## 遇到的问题和解决
1. 问题：FastAPI项目运行报ModuleNotFoundError: No module named 'app'，模块路径识别失败
解决：在main.py顶部添加sys.path.append配置，将项目根目录加入Python模块搜索路径，使用python -m app.main模块方式启动服务，解决路径问题。
2. 问题：Python低版本运行报TypeError: unsupported operand type(s) for |: 'type' and 'type'
解决：将models.py中dict | list | str联合类型，替换为typing.Union兼容写法，导入Union并修改为Union[dict, list, str]，适配低版本Python。
3. 问题：Apifox配置Mock后请求报Invalid URI无效地址错误
解决：配置正确的Mock环境变量base_url，接口路径使用{{base_url}}/api/xxx格式，拼接Apifox云端Mock专属域名，解决地址拼接错误。
4. 问题：后端API调用时出现请求参数与接口定义不匹配的问题
解决：对照OpenAPI文档规范，统一请求参数的命名和数据类型，封装标准化的API调用函数，做参数校验和格式转换，保证调用一致性。
